# 实现功能

+ 基本的聊天功能: **离线消息、群聊功能**

+ 基本的好友管理、个人信息管理功能: **添加好友**

+ 基本的文件和图片发送功能: **缓存式发送**

+ TIM一体化风格的聊天界面: **聊天气泡样式**

+ 基本的特殊消息: **窗口抖动**, **截屏**

# 项目亮点

+ **MVC架构**, **封装完备**, **全面向对象**, **可维护性强**

+ **用户界面友好**, **聊天体验优秀**
